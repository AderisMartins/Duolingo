## Projeto criado visando uma app parecida com Duolingo feito em Angular 7

### 1º Clone o Projeto
 git clone git@github.com:AderisMartins/Duolingo.git

### 2º Baixe as dependências
 npm install

### 3º Rode o projeto
 npm start
 
### 4º Abra o app usando a url no seu navegador
 http://localhost:4200/
